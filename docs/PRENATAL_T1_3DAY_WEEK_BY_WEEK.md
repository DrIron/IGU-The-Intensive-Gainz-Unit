# Prenatal Trimester 1 (3 Day) — Week-by-Week Build Sheet

Source-of-truth breakdown of the Ganbaru Method *Prenatal Trimester 1 (3 Day)* program for rebuilding in drafts. Source coach: Sheridan Skye. Captured from the Ganbaru iOS app, mapped to IGU `exercise_library` names (all exercises now exist in the library).

## Key facts

- **8 program weeks** (Week 1–8), 3 training days/week (Lower / Upper / Whole Body). These are program weeks; they map to pregnancy weeks 5–12 (Program Week 1 = pregnancy week 5; program week = pregnancy week − 4).
- **Phase 1 = Program Weeks 1–4** (pregnancy wks 5–8). **Phase 2 = Program Weeks 5–8** (pregnancy wks 9–12).
- Within a phase, **every week runs the same 3 session templates** — exercises do NOT change week to week. Progression is autoregulated: **every working set is prescribed at 3 RIR**, athlete adds load/reps as able. Exercises change only at the phase boundary (end of Week 4 → Week 5).
- **Rest** is per-exercise, in seconds (as prescribed by Ganbaru).
- **Supersets:** rows sharing a tag (SS1, SS2) are performed back-to-back.
- **Carries** are timed holds (30s per side), no RIR.
- Client logging: weight + reps per set (default).
