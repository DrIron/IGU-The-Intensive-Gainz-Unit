5 · What breaks coherence today (the backlog this creates)

Two editors

Planning Board vs legacy ProgramEditor / ProgramCalendarBuilder. → one board on plan*, retire legacy.

Saved ≠ draft look

Library/detail/macro don't use the builder's ribbon/tiles. → shared primitives (COACH_PROGRAMS_VIEW).

Duplicate rows

Drafts + Mesocycles = two copies of one thing. → one Programs tab, status pill.

studio/ alt-render

A second board look. → fold in or drop; don't maintain two.

Assign buried

Kebab items + silent skip. → primary Assign + the agreed dialog.

Desktop-only

Coaches program on phones. → mobile pass across every surface.
