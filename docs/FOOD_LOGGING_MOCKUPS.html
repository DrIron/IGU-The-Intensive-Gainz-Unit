IGU — Food Logging · mockups (v4 — 11 screens · themed, default dark)

One NutritionSummary everywhere: a macro-split donut with the calorie number centered inside, and a fixed-column legend — swatch · name · grams · % — that stays on one line and lines up row to row. The persistent client bottom nav (Dashboard · Nutrition · Workout · Learning · More) is always present; search & food detail are drawers over the Nutrition page, so the nav never disappears.

Fixed from v2: the legend no longer wraps/stacks (124, "/", "172g", stray "%" on top of each other) — every value now sits in its own aligned column. Calorie number in the donut is smaller and truly centered. Dashboard card is now Food diary and the log action is the + (same as the diary's), and it links out to history & trends. Adherence = a calorie band (adherent ±10% / off ±20%, hybrid: fixed now, coach-overridable later); macro alerts are two-tier — loud for protein + calories (banner + coach/client notification), quiet inline note for fat + carbs (see coach screen 6 + dashboard 8). New in v4: 9 History & Trends (intake-vs-target trend, adherence dot strip, macro trends), 10 dietitian meal-plan builder (build a meal by browsing food categories, like a workout), 11 admin Food Library Manager (curate the shared catalog: source + approval + provenance + promote client foods).
