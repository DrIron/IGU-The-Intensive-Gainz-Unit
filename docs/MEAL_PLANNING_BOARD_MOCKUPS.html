IGU — Meal Planning Board · mockups (v2 · themed, default dark)

Built like the workout board (Day → block → slot). Each slot is target-driven — the dietitian sets the amount they want (e.g. 40 g protein, or by kcal / carbs / fat) and the board auto-calculates the grams of each food to hit it (no manual math). A slot is either Pick one (options are alternatives, each auto-sized to the target — client picks one) or left Open (client chooses freely). Applies to meals & snacks; supplements deferred for now.

Changed from v1: options no longer need a hand-typed amount — set a macro/kcal target per slot and amounts compute automatically. Slots are Pick one or Open (Combine-all removed). Supplements removed from the board (revisit later). The rail still shows estimated daily kcal/macros + micros + the phase over-target alert.
